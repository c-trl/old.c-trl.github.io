Hypothesis: The more red in the logo, the more green in the bank. (If you sell food, that is.)

The idea of colors influencing how people behave has always been fascinating to me. One of the ways I wanted to approach exploring this idea was to look at what kind of presence the color red has in the restaurant industry. Supposedly, red makes you hungry (and apparently warmer monitor colors are easier on the eyes, according to various sources I've seen like this one), so if this is true, I would venture that restaurants that feature red in their marketing more should be more successful.

This project has been something stuck on the proverbial drawing board for quite some time. The first go at this experiment utilizes matplotlib, numpy, and pandas, hoewver the project stopped short after a few returned errors. This project has been put on hold until I learn more about what SciPy might be able to do, and about image resolution and color processing.
